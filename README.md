# 📦 Нормализация данных доставок и построение витрин (Airflow ETL/ELT)

### 📌 Описание задачи

Вам представлены денормализованные данные по доставкам.

Вам необходимо нормализовать данные, объяснив выбор нормальной формы и общие правила перехода, и с помощью ETL/ELT-процессов в Airflow сперва загрузить нормализованные данные в конечные таблицы, а затем построить 2 витрины, на основе которых аналитики смогут строить табличные и BI-отчеты.


#### Искомые метрики
Вам необходимо построить 2 витрины, на основе которых можно писать простые запросы (фильтры и агрегации), для расчета следующих метрик.
#### Метрики
Витрина заказов  
Оборот (на какую сумму заказали с учетом скидок)  
Выручка (какую сумму оплатили с учетом скидок и замен)  
Прибыль (выручка - расходы)  
Количество сделанных заказов (созданных)  
Количество доставленных заказов  
Количество отмененных заказов  
Количество отмен после доставки  
Количество отмен из-за ошибок на стороне сервиса (для причин отмены "Ошибка приложения" и "Проблемы с оплатой")  
Количество покупателей  
Средний чек  
Количество заказов на покупателя  
Выручка на покупателя  
Количество смен курьеров (заказы со сменой)  
Количество активных курьеров  

#### Витрина товаров
Оборот товара  
Количество заказанных единиц товара  
Количество отмененных единиц товара  
Количество заказов с товаром  
Количество заказов с отменой товара  
Самый популярный товар за день/неделю/месяц  
Самый непопулярный товар за день/неделю/месяц
