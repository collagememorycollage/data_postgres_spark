📦 Нормализация данных доставок и построение витрин (Airflow ETL/ELT)

📌 Описание задачи

Вам представлены денормализованные данные по доставкам.

Вам необходимо нормализовать данные, объяснив выбор нормальной формы и общие правила перехода, и с помощью ETL/ELT-процессов в Airflow сперва загрузить нормализованные данные в конечные таблицы, а затем построить 2 витрины, на основе которых аналитики смогут строить табличные и BI-отчеты.
